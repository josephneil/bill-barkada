# Bill Barkada

Bill Barkada is a small Next.js app for splitting group bills. Add people,
enter receipt items, choose who shared each item, apply service charge or a
discount, and copy a clean payment summary for chat.

## Features

- Add and remove people.
- Add receipt items with prices.
- Toggle which people shared each item.
- Split service charge and discounts proportionally.
- Save the current bill locally in the browser.
- Copy a shareable payment summary.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks
